<h1 align="center"> KEYBINDS </h1>

ESC: Quit


RED SPACESHIP:

  W: Up

  A: Left

  S: Down

  D: Right


  LCTRL: Fire


YELLOW SPACESHIP:

  ↑: Up

  ←: Left

  ↓: Down

  →: Right


  RCTRL: Fire
